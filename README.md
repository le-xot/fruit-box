# Development

## Dependencies

* [Node.js: v22](https://nodejs.org/en)
* [PNPM: v10](https://pnpm.io/)

## Cli

* Install dependencies

```bash
  pnpm i
```

* Start development

```bash
  pnpm dev
```

* Visit [http://localhost:5173](http://localhost:5173)

## Deployment

* If you want use github-pages:

```bash
  pnpm run deploy
```
